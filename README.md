## Field Operations `// selected projects`

> **MFHSS — Multi-Frequency-Hopping Spread-Spectrum Swarm**
> Adaptive anti‑jamming frequency‑hopping engine for a simulated drone swarm. Spectrum‑sensing, jammer, and adaptive hopset modules over a multi‑node UDP mesh (HQ · master · leader · drones) with live PSD/packet visualization. Same engine also drives a generative‑music demo, *Swarm Symphony*.
> `Python` · `NumPy` · `matplotlib`

> **SANDMAP-X — Tactical Sand-Table Simulation Engine**
> Deterministic, offline mission‑planning & rehearsal platform. C++20 core engine (logging, config, map loader, database) with a TypeScript/React shell and Python asset tooling. CMake + Ninja build, SHA256‑pinned deps, cross‑module tests.
> `C++20` · `TypeScript` · `React` · `Python`

> **Aspects of Life** — *founder / builder*
> A storytelling platform preserving real experiences and perspectives from around the world. **77 stories from 77 contributors** and counting. Designed the identity, built the site, shaped how human stories outlive a conversation.

> **Ferri — PM Translation Diagnostic**
> React 19 + TypeScript + Vite SPA measuring how well a product manager's judgment transfers across industries. Scored probes, live feedback, gated report preview.
